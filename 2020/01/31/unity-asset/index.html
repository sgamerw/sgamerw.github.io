Unity插件及资源使用记录

Jan 31, 2020

知道各个插件能干什么，可以提高效率，所以在这里记录一下使用过的插件的功能及体验。

Odin - Inspector and Serializer

号称可以序列化一切，各种方便的标签扩展编辑器功能，必备。

Entitas

ECS 框架，GitHub Wiki 界面有很多文档教程，YouTube 还有作者介绍的对应单元测试方法，概念清晰，数据可视化，Gitter 上面的用户社区也很活跃，唯一不足的是，作者似乎太忙了，已经很久没有更新过插件版本了。

DoTween

这个应该是用的最多的动作插件了吧。

Doozy UI

完善的 UI 管理框架，可视化编辑 UI 界面跳转的流程图，自带的多种界面出现消失的动画效果。

UI Framework

简洁的开源 UI 管理框架，结构，概念清晰，源码学习也是一个很好的选择。

Json.NET for Unity

与自定义类数据结构进行转换。

Easy Save

自带序列化方法，存储多种结构数据。

Procedural UI Image

程序化给图片设置圆角，非常适合制作扁平风格的 UI，适合独立开发者只用颜色就能制作简洁的 UI 了。

Safe Area Helper

全面屏屏幕适配解决方案，配合这篇文章，了解这个插件的工作方法，非常简洁。

Editor Console Pro

调用栈预览，自定义 tag，真机日志局域网发送到 Editor 进行查看，是内置 console 的替代方案。

I2 Localization

非常强大的多语言处理方案。

Log Viewer

可以在设备上查看日志信息，只需要画一个圈就行了，也是一个开源项目。

FancyScrollView

通过自定义列表中的元素的动画来制作各种效果的列表，是制作卡片式的列表的不二选择，也是一个开源项目。
